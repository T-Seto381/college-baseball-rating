大学野球試合結果データ収集に関する相談ログ
作成日: 2026-04-30
対象テーマ: 神宮への道の欠損期間（2019年全日本大学野球選手権〜2021年明治神宮大会）の大学野球試合結果収集


============================================================
1. ユーザーの目的
============================================================

ユーザーは、日本国内の大学野球の試合結果を集めている。

主に欲しいデータは以下。

- 試合日
- 対戦した2チーム
- 両チームの得点

既存の主データソースとして「神宮への道」を利用している。

神宮への道には以下の2系統がある。

- 旧サイト: https://univbbl.web.fc2.com
  - 2019年春までのデータがある
- 新サイト: https://www.univbbl.com
  - 2022年春から現在までのデータがある

例:
https://www.univbbl.com/tohoku/sendai6/2026/spring.html

このURLは、仙台六大学野球の2026年春シーズンの結果ページであり、
URLの地域・リーグ・年度・季節部分を加工することで任意のリーグ・時期のページを参照できる。

問題は、以下の期間が神宮への道だけでは埋まらないこと。

- 2019年の春の全国大会
- 2019年秋リーグ
- 2020年春リーグ
- 2020年秋リーグ
- 2021年春リーグ
- 2021年秋リーグ
- 2021年明治神宮大会

つまり、
「2019年春の全国大会〜2021年神宮大会」までの
全国の大学野球リーグの試合結果を取得したい。


============================================================
2. 提案した主なデータソース
============================================================

欠損期間を補う候補として、以下を提案した。

------------------------------------------------------------
2-1. 第一候補: 一球速報.com / OmyuTech
------------------------------------------------------------

最も現実的な主データソースとして提案。

理由:
- 大学野球カテゴリがある
- 大会・リーグごとに日程・結果・勝敗表・成績ページがある
- 2019〜2021年の一部大会・リーグ戦が確認できる
- cupId / leagueId ベースでページを機械的に辿れる可能性がある

URL形式の例:

https://baseball.omyutech.com/CupHomePageMain.action?cupId=20210000038
https://baseball.omyutech.com/leagueCup.action?leagueId=336

考え方:
1. leagueCup.action?leagueId=... で各連盟の大会一覧を取得
2. 2019秋、2020秋、2021春、2021秋などに該当する cupId を抽出
3. CupHomePageMain.action?cupId=... から日付・チーム・スコアを抽出
4. date=YYYYMMDD が付く日別URLがある場合は、それも巡回する

注意点:
- OmyuTechはページ構造が揺れる
- 静的HTMLだけで取れない場合がある
- rvestで取れなければ requests + BeautifulSoup、またはRSelenium / Playwrightが必要になる可能性がある
- 途中中止、不戦勝、没収試合などの例外処理が必要


------------------------------------------------------------
2-2. 第二候補: 全日本大学野球連盟 JUBF
------------------------------------------------------------

JUBFは、主に以下の用途で使うべきと提案。

- 全日本大学野球選手権大会の試合結果取得
- 加盟各連盟の公式サイトへのリンク集として利用
- 各リーグの開催状況確認

全国大会:
- 2019年 第68回全日本大学野球選手権大会
- 2021年 第70回全日本大学野球選手権大会
- 2020年 第69回全日本大学野球選手権大会は中止

JUBFのリーグ戦ページ:
- 2021年春季リーグ戦ページ
- 2021年秋季リーグ戦ページ

これらには各連盟の公式サイトリンクがまとまっているため、
OmyuTechにないリーグを各公式サイトから補完する際のハブとして使える。


------------------------------------------------------------
2-3. 第三候補: 日本学生野球協会
------------------------------------------------------------

明治神宮大会については、日本学生野球協会の公式ページを使うのが確実と提案。

============================================================
3. 2020年春の扱い
============================================================

2020年春は新型コロナウイルスの影響で多くのリーグ戦・全国大会が中止または変則開催。

扱いとしては以下のように整理。

- 2019春全国大会〜2019秋: 通常に近い
- 2020春: 多くが中止。存在しない試合が多い
- 2020秋: 変則開催・短縮開催あり
- 2021春: 開催あり。ただし地域差あり
- 2021秋〜神宮: 開催あり。ただし延期・方式変更あり

データベース上は、2020春について
「試合結果が欠損している」のではなく
「大会・リーグ自体が中止で試合が存在しない」
ケースを区別する必要がある。


============================================================
4. ユーザーから依頼された作業
============================================================

ユーザーから、以下の3つのコード作成依頼があった。

1. OmyuTechで対象期間の試合結果をスクレイピングするコードを作成する
2. 全国大会の試合結果をJUBFと日本学生野球協会からスクレイピングするコードを作成する
3. OmyuTechのないリーグをまとめて、各連盟公式サイトを開くコードを作成する

対象期間は、
2019年全日本大学野球選手権〜2021年明治神宮大会
として扱い、コード上ではおおむね以下に設定。

2019-06-01 ～ 2021-11-30


============================================================
5. 作成したRコードの構成
============================================================

Rでコピペ運用できる形として、以下の構成を提案した。

------------------------------------------------------------
5-0. 共通準備
------------------------------------------------------------

使用ライブラリ:
- tidyverse
- rvest
- httr2
- stringr
- lubridate
- janitor
- glue
- readr
- fs

作成した共通関数:
- safe_read_html()
  - URLを読み込む
  - User-Agentを付与
  - エラー時はNULLを返す
- clean_lines()
  - HTMLテキストを行単位に分割後、空白や全角スペースを整形
- parse_score()
  - "1 - 6" のようなスコア文字列を score1, score2 に変換
- parse_mmdd_to_date()
  - "6/10" と年を受け取り Date に変換


------------------------------------------------------------
5-1. OmyuTech cupId URL収集
------------------------------------------------------------

目的:
OmyuTechの leagueCup ページや既知の CupHomePageMain ページから、
対象となる cupId URLを収集する。

作成した関数:
- collect_omyutech_cup_urls(seed_urls)

入力:
- seed_urls
  - OmyuTechの leagueCup URL
  - または既知の CupHomePageMain URL

処理:
1. seed URLを読む
2. ページ内の a タグを取得
3. CupHomePageMain.action?cupId=... を含むURLを抽出
4. cupIdを取得
5. seed自体がcupページの場合も追加
6. 各cup URLのtitleを取得
7. data_out/omyutech_cup_urls.csv に保存

例として入れたseed URL:
- 関甲新学生野球連盟 leagueCup
- 2021 関甲新秋 1部
- 2021 関西学生秋
- 2021 関西六大学秋
- 2019 全日本大学選手権
- 2021 神宮大会 大学の部


------------------------------------------------------------
5-2. OmyuTech 試合結果パーサー
------------------------------------------------------------

目的:
OmyuTechのcupページ・日別ページから
試合日、チーム名、得点を抽出する。

作成した関数:
- is_scoreboard_line()
  - スコアボード行か判定
- scoreboard_total()
  - スコアボード行の最後の数字を最終得点として取得
- extract_omyutech_dates()
  - cupページ内から date=YYYYMMDD を含む日別URLを取得
- scrape_omyutech_one_page()
  - 1ページから試合結果を抽出
- scrape_omyutech_cup()
  - cupページ単位で日別URLを巡回して試合結果を抽出

基本ロジック:
以下のような並びを検出する。

チーム名
スコア行
チーム名
スコア行

スコア行の最後の数字を最終得点として使う。

出力ファイル:
- data_out/omyutech_game_results_201906_202111.csv
- logs/omyutech_failed_pages.csv

失敗時のnote:
- read_html_failed
- cup_read_failed
- no_game_parsed

注意:
OmyuTechはHTML構造が揺れるため、失敗ページをログに出して後から個別対応する設計。


------------------------------------------------------------
5-3. JUBF 全日本大学野球選手権スクレイピング
------------------------------------------------------------

目的:
JUBFから全日本大学野球選手権の試合結果を取得する。

対象年:
- 2019
- 2020
- 2021

作成した関数:
- scrape_jubf_alljapan(year)

URL形式:
https://www.jubf.net/system/prog/schedule.php?k=all&m=pc&s={year}

抽出する情報:
- source
- tournament
- year
- source_url
- date
- venue
- time
- round
- team1
- team2
- score1
- score2
- note

出力:
- data_out/jubf_alljapan_results_2019_2021.csv

2020年は中止の可能性をnoteに記録する設計。


------------------------------------------------------------
5-4. 日本学生野球協会 明治神宮大会スクレイピング
------------------------------------------------------------

目的:
日本学生野球協会から明治神宮大会 大学の部の試合結果を取得する。

対象年:
- 2019
- 2020
- 2021

作成した関数:
- scrape_jsba_jingu_college(year)

URL形式:
https://www.student-baseball.or.jp/system/prog/schedule.php?e=jingu&k=all&m=pc&s={year}

処理:
1. ページ本文を行単位に整形
2. 日付行を検出
3. 「大学」ブロックに入る
4. 第n試合、時刻、ラウンド、team1、score、team2を抽出
5. scoreが "2 - 1" 形式なら得点に変換
6. 高校ブロックは無視

出力:
- data_out/jsba_jingu_college_results_2019_2021.csv


------------------------------------------------------------
5-5. 全国大会結果の結合
------------------------------------------------------------

JUBFと日本学生野球協会の結果をbind_rowsで結合。

出力:
- data_out/national_tournament_results_2019_2021.csv


------------------------------------------------------------
5-6. JUBF 加盟連盟公式サイト一覧取得
------------------------------------------------------------

目的:
JUBFのリーグ戦ページから、各加盟連盟の公式サイトURLを取得する。

作成した関数:
- scrape_jubf_league_links(season_code)

URL形式:
https://www.jubf.net/system/prog/league_match_index.php?s={season_code}

season_code例:
- 2019a
- 2020s
- 2020a
- 2021s
- 2021a

ただし古いページが存在しない場合もあるため、
実用上は 2021s / 2021a を中心に使う想定。

出力:
- data_out/jubf_league_official_links.csv


------------------------------------------------------------
5-7. OmyuTech収集済みリーグ名の推定
------------------------------------------------------------

目的:
OmyuTechで既に収集できているリーグ名をcup titleから推定する。

出力:
- data_out/omyutech_league_guess.csv

注意:
文字列処理による推定なので、完全一致は期待しない。
最終的には目視確認前提。


------------------------------------------------------------
5-8. OmyuTechにないリーグの抽出
------------------------------------------------------------

目的:
JUBF公式サイト一覧とOmyuTech収集済みリーグ名を照合し、
OmyuTechにない可能性がある連盟を抽出する。

使用ライブラリ:
- stringdist

作成した関数:
- match_omyutech_league()

照合方法:
- Jaro-Winkler距離
- 閾値 0.35

出力:
- data_out/missing_omyutech_leagues.csv


------------------------------------------------------------
5-9. OmyuTechにない連盟公式サイトをブラウザで開く
------------------------------------------------------------

目的:
missing_omyutech_leagues に含まれる公式サイトURLをブラウザで開く。

作成した関数:
- open_missing_league_sites(missing_tbl, n = 5, start = 1)

仕様:
- 一気に大量に開くとブラウザが重くなるため、5件ずつ開く設計
- browseURL() を使用

例:
open_missing_league_sites(missing_omyutech_leagues, n = 5, start = 1)
open_missing_league_sites(missing_omyutech_leagues, n = 5, start = 6)

============================================================
7. 実行順の整理
============================================================

実行順は以下。

1. collect_omyutech_cup_urls(seed_urls)
   - OmyuTech cupId収集

2. scrape_omyutech_cup()
   - OmyuTech試合結果収集

3. scrape_jubf_alljapan()
   - JUBF 全日本大学野球選手権収集

4. scrape_jsba_jingu_college()
   - 日本学生野球協会 明治神宮大会収集

5. scrape_jubf_league_links()
   - JUBFから各連盟公式サイト一覧取得

6. OmyuTechリーグ名推定

7. JUBF公式一覧とOmyuTech収集済みを突合

8. open_missing_league_sites()
   - OmyuTechにない連盟公式サイトを順番に開く

============================================================
9. 現時点の結論
============================================================

欠損期間の大学野球試合結果収集は、以下の方針が最も現実的。

1. OmyuTechを主データソースにする
2. 全国大会はJUBFと日本学生野球協会から公式データを取る
3. OmyuTechにないリーグは、JUBFの加盟連盟リンク集から各連盟公式サイトを確認する
4. 取れなかったページをログ化し、追加パーサーで潰していく

完全自動で全国全リーグを一発取得するのは難しいが、
「cupId巡回 + 失敗ログ + 公式サイト補完」の設計なら、
実務的にかなり高い割合で回収できる。
